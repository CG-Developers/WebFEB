# Fotos per a la web / Fotos para la web

Afegeix les teves fotos en aquesta carpeta amb els noms exactes indicats.
Quan existeixi el fitxer, apareixerà automàticament a la web.

## Fotos necessàries

### Home (index.html)
- `hero-home.jpg` — Hero principal (recomanat: 1920×1080px, apaïsada, flamenc en escena)
- `about-home.jpg` — Secció "qui som" a la home (retrat o escena)

### Tablao (tablao.html)
- `hero-tablao.jpg` — Hero de la pàgina tablao
- `tablao-show-1.jpg` — Foto espectacle 1
- `tablao-show-2.jpg` — Foto espectacle 2
- `tablao-show-3.jpg` — Foto espectacle 3

### Talleres (talleres.html)
- `hero-talleres.jpg` — Hero de tallers
- `talleres-1.jpg` — Foto taller grup
- `talleres-2.jpg` — Foto taller empresa

### Clases (clases.html)
- `hero-clases.jpg` — Hero de classes
- `clases-1.jpg` — Foto classe

### Galeria (galeria.html)
- `galeria-01.jpg` fins a `galeria-12.jpg` — 12 fotos per la galeria
- (opcional) `galeria-video-thumb.jpg` — Miniatura vídeo

### Qui som (sobre-nosotros.html)
- `hero-about.jpg` — Hero de la pàgina
- `equipo-1.jpg` — Foto persona 1
- `equipo-2.jpg` — Foto persona 2
- `equipo-3.jpg` — Foto persona 3

### Blog (blog.html)
- `blog-01.jpg`, `blog-02.jpg`, `blog-03.jpg` — Miniatures articles

## Format recomanat
- JPG, qualitat 80–85%
- Mida mínima: 1200px d'amplada per fotos hero
- Mida galeria: 800×600px
- Fotos equip: 600×800px (retrat)
